Algorithms

Most searching algorithms in genetics start with string searches of a small (<100char) string within a much larger (>1Gchar) string. These can be facilitated with a Suffix Array (SA) which is a sorted array of substrings within the string.

For example, SA("hello") to {"", "ello", "hello", "llo", "lo", "o"}.
